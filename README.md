# Reflexive Vault - Self-Defending AI Agent

## Features
- Cryptographic Identity (Ed25519 DIDs)
- Dynamic Trust Scoring (Immune System)
- Agent-to-Agent Trust Registry
- Step-up Authentication for HIGH-risk
- Complete Audit Logging
- Free AI via ClawRouter (no credit card)

## How to Run
1. Install: `pip install fastapi uvicorn requests`
2. Run ClawRouter: `npx @blockrun/clawrouter`
3. Run server: `python web_server.py`
4. Open `agent.html` in browser
